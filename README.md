# Aircraft Component Life Monitoring System

## Overview

The Aircraft Component Life Monitoring System is an aviation-focused maintenance planning tool developed using Microsoft Excel.

The purpose of this project is to assist maintenance planners and airworthiness personnel in tracking aircraft component life limits based on:

* Flight Hours (FH)
* Flight Cycles (FC)
* Calendar Days

The system automatically evaluates remaining life and identifies components requiring attention.

---

## Problem Statement

Aircraft components are often controlled by multiple maintenance limits.

Tracking these limits manually can be time-consuming and may increase the risk of oversight.

This tool provides a structured method for monitoring component status and prioritizing maintenance actions.

---

## Features

* Flight Hour tracking
* Flight Cycle tracking
* Calendar-based monitoring
* Remaining life calculation
* Maintenance urgency indication
* Maintenance planning support
